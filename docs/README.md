## 🚀 技术栈概览

- **后端框架:** NestJS 9.x + TypeScript 4.x
- **数据库:** MySQL 8.0 + TypeORM
- **认证:** JWT + Guards
- **文档:** Swagger/OpenAPI 3.0
- **云服务:** 火山引擎 (翻译API + TOS存储)
- **运行端口:** 3010
